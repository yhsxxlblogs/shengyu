# 声愈项目部署文档

## 一、环境要求

### 服务器配置建议

| 配置项 | 最低配置 | 推荐配置 |
|--------|----------|----------|
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 硬盘 | 50GB SSD | 100GB SSD |
| 带宽 | 5Mbps | 10Mbps |
| 操作系统 | CentOS 7+/Ubuntu 18.04+ | CentOS 8/Ubuntu 20.04 |

### 软件环境

| 软件 | 版本 | 说明 |
|------|------|------|
| Node.js | 16.x/18.x LTS | 后端运行环境 |
| MySQL | 5.7+/8.0 | 关系型数据库 |
| Redis | 5.0+ | 缓存数据库 |
| Nginx | 1.18+ | 反向代理/Web服务器 |
| PM2 | 最新版 | Node.js进程管理 |
| Git | 2.x | 版本控制 |

```bash
sudo firewall-cmd --permanent --add-port=443/tcp
sudo firewall-cmd --permanent --add-port=3000/tcp

sudo iptables -I INPUT -p tcp --dport 3000 -j ACCEPT
```

## 四、安装MySQL

### 安装

```bash
# Ubuntu
sudo apt install -y mysql-server

# CentOS
sudo yum install -y mysql-server
sudo systemctl start mysqld
```

### 安全配置

```bash
# 运行安全脚本
sudo mysql_secure_installation

# 根据提示设置：
# - 设置root密码
# - 移除匿名用户
# - 禁止root远程登录
# - 移除测试数据库
# - 重新加载权限表
```

### 创建数据库

```bash
# 登录MySQL
sudo mysql -u root -p

# 创建数据库
CREATE DATABASE animal_sound_app CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 创建应用用户
CREATE USER 'shengyu'@'localhost' IDENTIFIED BY 'YourStrongPassword123!';

# 授权
GRANT ALL PRIVILEGES ON animal_sound_app.* TO 'shengyu'@'localhost';
FLUSH PRIVILEGES;

# 退出
EXIT;
```

### 配置远程访问（可选）

```bash
# 编辑MySQL配置
sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf

# 注释掉bind-address
# bind-address = 127.0.0.1

# 重启MySQL
sudo systemctl restart mysql
```

## 五、安装Redis

### 安装

```bash
# Ubuntu
sudo apt install -y redis-server

# CentOS
sudo yum install -y epel-release
sudo yum install -y redis
```

### 配置

```bash
# 编辑Redis配置
sudo vim /etc/redis/redis.conf

# 修改以下配置
# 设置密码
requirepass YourRedisPassword123!

# 允许远程访问（如需）
# bind 0.0.0.0

# 重启Redis
sudo systemctl restart redis
sudo systemctl enable redis
```

### 验证

```bash
# 连接Redis
redis-cli

# 认证
AUTH YourRedisPassword123!

# 测试
PING
# 应返回 PONG

# 退出
EXIT
```

### 缓存策略配置

项目已内置缓存雪崩、击穿、穿透的防护机制，无需额外配置：

#### 1. 缓存雪崩防护
- 随机过期时间（5-10分钟）
- 定时任务每5分钟更新热门数据

#### 2. 缓存击穿防护
- 热点数据永不过期（定时更新）
- 互斥锁防止并发重建（可选）

#### 3. 缓存穿透防护
- 空值缓存（空数组缓存60秒）
- 参数校验过滤非法请求

**注意**：确保Redis内存充足，建议至少512MB

## 六、安装Nginx

### 安装

```bash
# Ubuntu
sudo apt install -y nginx

# CentOS
sudo yum install -y nginx
sudo systemctl start nginx
```

### 配置Nginx

```bash
# 创建配置文件
sudo vim /etc/nginx/conf.d/shengyu.conf
```

添加以下内容：

```nginx
# HTTP服务 - 后端API
server {
    listen 80;
    server_name your-domain.com;

    # 静态文件
    location /uploads {
        alias /var/www/shengyu-backend/uploads;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # 管理后台
    location /admin {
        alias /var/www/shengyu-backend/admin;
        index index.html;
        try_files $uri $uri/ /admin/index.html;
    }

    # API代理
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        
        # 超时设置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    # WebSocket代理
    location /ws {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 86400;
    }
}

# HTTPS配置（使用SSL证书后启用）
# server {
#     listen 443 ssl http2;
#     server_name your-domain.com;
#     
#     ssl_certificate /path/to/cert.pem;
#     ssl_certificate_key /path/to/key.pem;
#     
#     # 其他配置同上
# }
```

### 测试并重载Nginx

```bash
# 测试配置
sudo nginx -t

# 重载配置
sudo systemctl reload nginx
```

## 七、部署后端服务

### 1. 克隆代码

```bash
# 创建应用目录
sudo mkdir -p /var/www
cd /var/www

# 克隆代码（请替换为实际仓库地址）
sudo git clone https://github.com/your-repo/shengyu-backend.git

# 修改权限
sudo chown -R $USER:$USER /var/www/shengyu-backend
```

### 2. 安装依赖

```bash
cd /var/www/shengyu-backend

# 安装依赖
npm install

# 生产环境依赖（可选）
npm install --production
```

### 3. 配置环境变量

```bash
# 创建环境变量文件
cp .env.example .env

# 编辑配置
vim .env
```

添加以下内容：

```bash
# 服务器配置
PORT=3000
WS_PORT=3001
NODE_ENV=production

# 数据库配置
DB_HOST=localhost
DB_USER=shengyu
DB_PASSWORD=YourStrongPassword123!
DB_NAME=animal_sound_app
DB_PORT=3306

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=YourRedisPassword123!

# JWT密钥（请修改为随机字符串）
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production

# 文件上传限制
MAX_FILE_SIZE=10485760
```

### 4. 初始化数据库

```bash
# 运行数据库初始化（如果有迁移脚本）
npm run db:migrate

# 或导入SQL文件
mysql -u shengyu -p animal_sound_app < database/backup_*.sql
```

### 5. 使用PM2启动服务

```bash
# 创建PM2配置文件
vim ecosystem.config.js
```

添加以下内容：

```javascript
module.exports = {
  apps: [
    {
      name: 'shengyu-api',
      script: './index.js',
      instances: 1,
      exec_mode: 'fork',
      env: {
        NODE_ENV: 'production',
        PORT: 3000
      },
      log_file: './logs/combined.log',
      out_file: './logs/out.log',
      error_file: './logs/error.log',
      log_date_format: 'YYYY-MM-DD HH:mm:ss Z',
      max_memory_restart: '500M',
      restart_delay: 3000,
      max_restarts: 10,
      min_uptime: '10s',
      watch: false,
      merge_logs: true,
      kill_timeout: 5000,
      listen_timeout: 10000
    }
  ]
};
```

```bash
# 创建日志目录
mkdir -p logs

# 启动服务
pm2 start ecosystem.config.js

# 保存PM2配置
pm2 save

# 设置开机自启
pm2 startup
# 执行输出的命令
```

### 6. PM2常用命令

```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs shengyu-api
pm2 logs shengyu-api --lines 100

# 重启服务
pm2 restart shengyu-api

# 停止服务
pm2 stop shengyu-api

# 重载服务（零停机）
pm2 reload shengyu-api

# 删除服务
pm2 delete shengyu-api
```

```bash
# 修改权限
sudo chown -R $USER:$USER /var/www/shengyu-app
```

### 2. 安装依赖并构建

```bash
cd /var/www/shengyu-app

# 修改API地址配置
vim utils/api.js
# 将 BASE_URL 修改为服务器地址

# 构建H5版本
npm run build:h5

# 或构建其他平台
# npm run build:mp-weixin  # 微信小程序
# npm run build:app        # App
```

### 3. 部署静态文件

```bash
# 创建Nginx静态文件目录
sudo mkdir -p /var/www/html/shengyu-app

# 复制构建文件
sudo cp -r dist/build/h5/* /var/www/html/shengyu-app/

# 修改Nginx配置添加前端服务
sudo vim /etc/nginx/conf.d/shengyu.conf
```

添加前端服务配置：

```nginx
# 前端H5服务
server {
    listen 8080;
    server_name your-domain.com;
    
    root /var/www/html/shengyu-app;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

```bash
# 测试并重载Nginx
sudo nginx -t
sudo systemctl reload nginx
```

## 九、配置HTTPS（SSL证书）

### 使用Let's Encrypt免费证书

```bash
# 安装Certbot
sudo apt install -y certbot python3-certbot-nginx

# 申请证书
sudo certbot --nginx -d your-domain.com

# 自动续期测试
sudo certbot renew --dry-run
```

### 手动配置SSL证书

```bash
# 编辑Nginx配置
sudo vim /etc/nginx/conf.d/shengyu.conf
```

添加HTTPS配置：

```nginx
server {
    listen 443 ssl http2;
    server_name your-domain.com;
    
    ssl_certificate /path/to/your/fullchain.pem;
    ssl_certificate_key /path/to/your/privkey.pem;
    
    # SSL优化
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers off;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 1d;
    
    # 其他配置...
}

# HTTP重定向到HTTPS
server {
    listen 80;
    server_name your-domain.com;
    return 301 https://$server_name$request_uri;
}
```

## 十一、备份策略

### 数据库备份脚本

```bash
# 创建备份脚本
vim /var/www/backup.sh
```

添加以下内容：

```bash
#!/bin/bash

# 配置
DB_NAME="animal_sound_app"
DB_USER="shengyu"
DB_PASS="YourStrongPassword123!"
BACKUP_DIR="/var/backups/shengyu"
DATE=$(date +%Y%m%d_%H%M%S)
RETENTION_DAYS=7

# 创建备份目录
mkdir -p $BACKUP_DIR

# 备份数据库
mysqldump -u$DB_USER -p$DB_PASS $DB_NAME | gzip > $BACKUP_DIR/db_backup_$DATE.sql.gz

# 备份上传文件
tar -czf $BACKUP_DIR/uploads_backup_$DATE.tar.gz -C /var/www/shengyu-backend uploads

# 删除7天前的备份
find $BACKUP_DIR -name "*.sql.gz" -mtime +$RETENTION_DAYS -delete
find $BACKUP_DIR -name "*.tar.gz" -mtime +$RETENTION_DAYS -delete

echo "Backup completed: $DATE"
```

```bash
# 添加执行权限
chmod +x /var/www/backup.sh

# 添加定时任务
crontab -e

# 每天凌晨3点执行备份
0 3 * * * /var/www/backup.sh >> /var/log/shengyu_backup.log 2>&1
```

## 十三、常见问题排查

```bash
# 检查端口占用
sudo netstat -tlnp | grep 3000
```

## 十五、安全加固

### 1. 禁用root登录

```bash
# 编辑SSH配置
sudo vim /etc/ssh/sshd_config

# 修改以下配置
PermitRootLogin no
PasswordAuthentication no  # 使用密钥登录

# 重启SSH
sudo systemctl restart sshd
```

### 2. 配置Fail2Ban

```bash
# 安装
sudo apt install -y fail2ban

# 配置
sudo vim /etc/fail2ban/jail.local
```

添加：

```ini
[DEFAULT]
bantime = 3600
findtime = 600
maxretry = 5

[sshd]
enabled = true
port = ssh
filter = sshd
logpath = /var/log/auth.log
```

```bash
sudo systemctl restart fail2ban
```

### 3. 定期安全更新

```bash
# 设置自动安全更新
sudo apt install -y unattended-upgrades
sudo dpkg-reconfigure unattended-upgrades
```
